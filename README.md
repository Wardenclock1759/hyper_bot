# Hypertension classifyer telegram bot
This program is a simple implementation of Telegram bot that classifies the possibility of having a condition based on certain values.
This bot is based on the following use case [Bot workflow](https://www.figma.com/file/T4n327es24e8D9GBv3dQMo/Telegram-Bot?type=design&node-id=0%3A1&mode=design&t=429wsIeLM41UZ9rn-1).
## How to run
To run this bot register new telegram bot using BotFather and connect server with dedicated chat.
